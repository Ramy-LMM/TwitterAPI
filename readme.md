# Twitter API のプログラム集
## このレポジトリはTwitter APIを使ったプログラムを保存しています。
バーチャルYoutuberをしている**ラミィ**です。<br>
<br>
このプログラムを使用するには、<br>
<br>
**Twitter APIキー**<br>
<br>と<br>
<br>
**Python**<br>
<br>
の環境が必要です。<br>
Pythonはダウンロードサイトからインストーラーをダウンロードしてきて、インストールしてください。<br>
[Pythonのダウンロードサイト](https://www.python.org/downloads/)<br>
<br>
インストールが完了したら、コマンドプロンプト上で<br>
```
pip install tweepy
```
を入力して下さい。<br>
これで環境は揃いました。<br>
